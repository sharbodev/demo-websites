# 💈 AI-Администратор для Барбершопа (Telegram Bot)

Умный Telegram-бот для автоматизации записи и консультирования клиентов барбершопа. Бот работает на базе языковой модели **Google Gemini** для свободных диалогов и содержит встроенный пошаговый мастер (FSM) для надежной записи на услуги.

## 🚀 Возможности

1. **ИИ-Консультант**: Отвечает на любые вопросы об услугах, ценах, мастерах, времени и месте работы. Не выходит из роли администратора барбершопа.
2. **Мастер Записи (Booking Wizard)**: Пошаговый выбор услуг, мастеров, даты и свободного времени с сохранением записи.
3. **Локальное хранилище**: Записи сохраняются локально в `bookings.json` (можно легко масштабировать на Supabase/PostgreSQL).
4. **Легкая настройка**: Контекст бизнеса полностью настраивается в файле `templates/barbershop.json`.

## 🛠️ Быстрый старт

### 1. Подготовка окружения и ключей

1. **Telegram Bot Token**:
   - Напишите [@BotFather](https://t.me/BotFather) в Telegram.
   - Создайте нового бота через команду `/newbot`.
   - Скопируйте полученный **API Token**.

2. **Google Gemini API Key**:
   - Перейдите в [Google AI Studio](https://aistudio.google.com/).
   - Создайте бесплатный API ключ.

### 2. Установка и запуск

1. Переименуйте `.env.template` в `.env` или откройте `.env` и введите ваши ключи:
   ```env
   TELEGRAM_BOT_TOKEN=ваш_токен_телеграм_бота
   GEMINI_API_KEY=ваш_ключ_gemini_api
   ```

2. Установите зависимости (рекомендуется использовать виртуальное окружение `venv`):
   ```bash
   # Создание venv
   python -m venv venv
   
   # Активация venv (Windows)
   venv\Scripts\activate
   
   # Активация venv (macOS/Linux)
   source venv/bin/activate
   
   # Установка библиотек
   pip install -r requirements.txt
   ```

3. Запустите бота:
   ```bash
   python main.py
   ```

## 📁 Структура проекта

- `main.py` — Точка входа, инициализация `aiogram` бота, обработчики команд и текстовых сообщений.
- `ai_engine.py` — Интеграция с Google Gemini API, управление историей диалогов и промптами.
- `booking.py` — Логика пошаговой записи (FSM) и клавиатуры.
- `config.py` — Загрузка настроек окружения и конфигурационного JSON ниши.
- `templates/barbershop.json` — База знаний барбершопа: контакты, услуги, цены, мастера, FAQ и инструкции для AI.
- `bookings.json` — Файл, куда сохраняются записи клиентов (создается автоматически).

## ☁️ Деплой на бесплатный хостинг

### Вариант: Render.com (Бесплатно)
1. Создайте аккаунт на [Render.com](https://render.com/).
2. Подключите ваш GitHub репозиторий с проектом.
3. Создайте новый **Background Worker** (поскольку боту не нужен открытый веб-порт, а запускается постоянное long-polling соединение).
4. Укажите параметры:
   - **Environment**: `Python`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python main.py`
5. В разделе **Environment Variables** добавьте:
   - `TELEGRAM_BOT_TOKEN`
   - `GEMINI_API_KEY`
6. Нажмите **Deploy**. Бот будет работать 24/7.
